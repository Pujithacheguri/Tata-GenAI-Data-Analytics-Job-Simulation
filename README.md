# Tata iQ – GenAI Powered Data Analytics Job Simulation

## Overview

Completed the Tata iQ GenAI Powered Data Analytics Job Simulation on Forage.

The project focuses on customer delinquency prediction using exploratory data analysis, predictive modeling, business recommendations, and responsible AI principles.

---

## Objectives

- Perform Exploratory Data Analysis (EDA)
- Identify customer delinquency risk factors
- Design a predictive machine learning solution
- Develop an AI-powered collections strategy
- Recommend ethical and responsible AI practices

---

## Tools & Technologies

- Microsoft Excel
- Machine Learning
- Random Forest
- Generative AI
- Business Analytics
- Microsoft PowerPoint

---

## Project Deliverables

- Exploratory Data Analysis Report
- Predictive Model Plan
- Business Summary Report
- AI-Powered Collections Strategy

---

## Key Insights

- High credit utilisation and repeated missed payments are strong indicators of delinquency.
- Credit score and debt-to-income ratio significantly influence repayment behaviour.
- Random Forest was selected as the predictive model for customer risk classification.
- Responsible AI principles improve fairness, transparency, and customer trust.
